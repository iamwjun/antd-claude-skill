# Ant Design Claude Skill - Examples

This directory contains complete working examples of common Ant Design patterns.

## Examples

### 1. Dashboard Layout
A complete dashboard with sidebar, header, and content area.

### 2. CRUD Operations
Full Create, Read, Update, Delete operations with forms and tables.

### 3. User Management
User list with search, filter, and pagination.

### 4. Data Visualization
Charts and graphs using Ant Design charts.

## How to Use

Each example is self-contained and can be copied directly into your project.
Make sure you have the required dependencies installed:

```bash
npm install antd @ant-design/icons
```

Import the component:

```tsx
import Example from './examples/ExampleName';
```
